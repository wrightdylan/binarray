ยงBinArray

Uses standard unsigned integers as statically sized binary arrays. This will have improved performance over the BitVecs dynamically sized binary vector. This is done by keeping the array out of heap allocation, improving locality, and uses lower cost functions. Having a static array also permits other optimisations. To further enhance performance, the functions do not provide error checking due to potential branch prediction misses. It is the respon- sibility of the programmer to ensure the index is within bounds, for example.
